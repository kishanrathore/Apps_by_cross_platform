#kishan_rathore

How To Use This Code  for install the app in Android
1.Download Or Clone the Code
2.Open Android Studio(You Must Have Setup flutter Sdk).
3.Open New flutter Project.
4.Change the lib/main.dart file with the downloaded one.
5.Change the Pubspec.yaml file with the downloaded one.
6.Copy the image folder into root folder of app.
7.Run the Code.
